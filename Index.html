Навыки

    Изучение CSS (Cascading Style Sheets) стало важным этапом в моем пути к освоению веб-разработки. С самого начала я понимал, что создание красивых и функциональных веб-страниц невозможно без знания стилей.

  • JavaScript

    JavaScript как и HTML я изучал в контексте веб-разработки. Я создавал простые веб-страницы и приложения, что дало мне представление о том, как работает клиентская часть.

    C# я изучал в рамках учебной программы и использовал его для разработки небольших приложений. Мне нравится его строгая типизация и мощные возможности для работы с объектно-ориентированным программированием.

  • Python

    Python как и C# я изучал в рамках учебной программы и использовал его для разработки небольших приложений. включая автоматизацию процессов и анализ данных. Его простота и читаемость кода позволяют мне быстро реализовывать идеи и прототипы.
